# Contributing

Thanks for your interest in gitos.

**How development works.** The canonical source of truth is a private repository; this public
repo is a one-way *published* mirror of the generic engine. Changes flow **private → public**,
never the reverse, through an allowlisted publish step.

**That means:** issues and pull requests are very welcome here, but a merged PR is integrated by
a maintainer **cherry-picking it into the private source**, from which it is re-published. Your
contribution keeps your authorship; it simply lands via that curated path rather than a direct
merge to this repo's history.

**Good first contributions:** clarifications to the role briefs (`references/`), additional
domain-adaptation examples in `references/anomaly-families.md`, and portability fixes in
`scripts/`.
